# Feature Lock and Variation

Separate reference facts into locked identity and permitted geometry before generating images.

## Locked Features

Record these as immutable unless the user explicitly overrides them:

| Dimension | Lock |
| --- | --- |
| Target boundary | Exact single prop or intended inseparable set |
| Category and function | What the object is and how it works |
| Era and style | Historical period, industrial language, stylization level |
| Components | Required parts and functional relationships |
| Palette | Main, secondary, accent colors and brightness hierarchy |
| Color placement | Which structural regions carry each color |
| Pattern | Motif content, sequence, orientation, density, and structural region |
| Materials | Ceramic, bronze, glass, leather, cloth, plastic, wood, etc. |
| Surface language | Glaze, grain, oxidation, scratches, stitching, dust, wear level |
| Signature details | Locks, hinges, badges, bands, motifs, fasteners, nose pads, feet |
| Symmetry | Required symmetry or deliberate asymmetry |

Patterns may stretch or compress naturally with a changed surface. Do not invent, delete, mirror, rotate, or move motifs to another structural region.

## Variable Features

Use geometry-only axes:

- Overall width/height ratio.
- Main silhouette and taper.
- Main body-to-base/rim/top proportion.
- Neck, mouth, crown, frame, lid, or shoulder proportion.
- Edge radius, corner treatment, and contour curvature.
- One major accessory shape or placement: handle, temple, brim, foot, latch zone, support, or mounting element.

Do not use color, texture, lighting, camera, crop, object rotation, decoration, or added components as a variation axis.

## Variation Strength

Default to medium controlled variation:

- Preserve immediate category and family recognition.
- Make silhouette differences readable in the contact sheet at thumbnail size.
- Preserve functional plausibility and component relationships.
- Avoid changing more than one primary and one secondary axis per derivative.

Light variation changes local proportions only. Strong variation may reorganize major masses and must be explicitly requested.

## Default Five-Variant Roles

| Variant | Role | Primary axis | Secondary axis |
| --- | --- | --- | --- |
| 01 | Standardized prototype | None | Cleanup only |
| 02 | Proportion derivative | Width/height and silhouette | Taper |
| 03 | Mass derivative | Main body-to-base/rim/top ratio | Transition position |
| 04 | Contour derivative | Edge/corner/curve language | Shoulder or side curvature |
| 05 | Accessory derivative | One major attached structure | Its mounting position |

Adapt the nouns to the category. For glasses, use lens/frame/bridge/temple geometry. For ceramics, use body/foot/neck/rim/handle. For cases, use body/corner/handle/latch zones.

## Rejection Rules

Reject:

- A different category, function, era, or style family.
- Missing or added functional components.
- Color, pattern, material, or wear drift.
- A variant distinguishable only by rendering, perspective, crop, or scale.
- An unplanned change to multiple major structures.
- Decorative novelty that masks weak geometry variation.
- A form that is implausible to manufacture, hold, wear, open, mount, or use.
